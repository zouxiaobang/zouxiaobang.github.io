hexo命令及markdown语法

Posted on 2016-08-26 | In Hexo

hexo命令

1、新建

hexo new "my blog article"

2、编译

hexo generate

或
hexo g
将md文件编译成html文件，存在public文件夹中

3、开启本地服务

hexo server

或
hexo s
开启后可通过该网址预览(http://localhost:4000)

4、部署到github中

hexo deploy

或
hexo d

5、清除public

hexo clean

当source文件夹中的资源有改动时就需要用到它
